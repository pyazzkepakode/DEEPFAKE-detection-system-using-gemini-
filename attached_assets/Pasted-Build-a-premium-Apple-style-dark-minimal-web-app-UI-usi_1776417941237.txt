Build a premium Apple-style dark minimal web app UI using HTML, CSS, and JavaScript.

App name: VeriSight

Goal:
Create a futuristic deepfake detection UI with smooth animations, scroll effects, and a premium Apple-like experience.

Design style:
- Apple-inspired (clean, minimal, elegant)
- Dark mode (#0f0f0f background)
- Glassmorphism (blurred translucent panels)
- Soft shadows and depth
- Rounded corners (16px+)
- Smooth transitions and animations
- Modern font (Inter or SF Pro style)

Layout & UX:

1. HERO SECTION (First screen):
- Full screen landing view
- Large bold title: "VeriSight"
- Subtitle: "AI-powered Deepfake Detection"
- Smooth fade-in + slide-up animation on load
- Slight parallax or motion effect
- Scroll indicator (arrow or hint to scroll)

2. SCROLL TRANSITION:
- Smooth scroll animation
- As user scrolls, hero fades out and next section slides in
- Add subtle motion effects (like Apple website)

3. MAIN APP SECTION (centered):
- Upload section should be perfectly centered
- Glassmorphic card container
- Drag & drop upload box
- Hover glow effect

4. INPUT PREVIEW:
- Show uploaded video with rounded corners
- Smooth fade-in animation

5. DETECT BUTTON:
- Gradient button
- Hover animation (glow or scale effect)
- Disabled until video is uploaded

6. LOADING STATE:
- Minimal spinner or pulse animation
- Smooth transition while processing

7. OUTPUT SECTION:
- Show processed video player
- Fade-in when ready

8. RESULT DISPLAY:
- Large text: REAL or FAKE
- REAL → green (#22c55e)
- FAKE → red (#ef4444)
- Show confidence %
- Smooth reveal animation

Functionality:
- Use JavaScript to:
  - Preview uploaded video
  - Send video via POST request:
    http://127.0.0.1:5000/predict
  - Receive processed video (blob)
  - Display it in output player
- Show loader while waiting

Code requirements:
- Separate HTML, CSS, JS files
- Clean and readable code
- No frameworks (vanilla JS)

Extra polish:
- Add smooth scrolling
- Add micro-interactions (hover, click)
- Add subtle animations (fade, slide, scale)
- Make UI feel like Apple product launch page

Final goal:
The website should feel premium, futuristic, smooth, and interactive — like an Apple product landing page combined with a functional AI tool.
